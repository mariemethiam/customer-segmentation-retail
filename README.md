# 🛍️ Customer Segmentation for CRM Optimization

## 🎯 Objective
Analyze customer behavior and build a segmentation model to improve CRM targeting and increase customer lifetime value.

---

## 📊 Dataset
Retail transactional dataset including:
- Customers  
- Orders  
- Products  
- Payments  

---

## ⚙️ Tools & Technologies
- Python (Pandas, NumPy, scikit-learn)  
- SQL  
- Power BI  

---

## 🔍 Analysis
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature engineering (RFM + behavioral metrics)  

---

## 🤖 Modeling
- K-Means clustering  
- Customer segmentation into 4 groups  

---

## 💡 Key Insights
- Identification of high-value customers  
- Detection of low-frequency / churn-risk segments  
- Behavioral differences between segments  

---

## 📈 Business Impact
- Improved CRM targeting  
- Personalized marketing strategies  
- Potential increase in campaign performance  

---

## 📊 Dashboard
Power BI dashboard to visualize customer insights
